# AWS Real-Time Stock Market Data Platform

This project implements an end-to-end real-time data engineering pipeline 
to ingest, process, and analyze live stock market data using modern 
big data technologies.

## Tech Stack
- Python
- Kafka
- Spark Structured Streaming
- AWS S3
- AWS Redshift
- Airflow
- Docker
- Power BI

## Architecture
Users -> API -> Kafka -> Spark -> S3 -> Redshift -> BI Dashboard
